# uwu-server

### Usage
```
> go install github.com/Toyz/uwu-server
> uwu-server
```

### Site usage
`> https://uwuify.helba.ai/?uwu=your text here`

### Nightbot usage
```
> !commands add !uwu $(urlfetch json https://uwuify.helba.ai/?uwu=$(querystring));
> !uwu This is my text
```
